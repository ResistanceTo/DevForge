## 🚀 安装

- **系统要求**：macOS 14.0 或更高版本

[![Mac App Store](./download-mas.svg)](https://apps.apple.com/app/id6752011791)

## 🌍 社区

你的反馈和贡献能让 DevForge 变得更好。

- **贡献代码**：欢迎提交 issue 或 pull request 来提出改进建议
- **赞助支持**：下载应用后，可以在应用内选择赞助方式支持开发

## 🙏 致谢
